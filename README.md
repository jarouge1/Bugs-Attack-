# Bugs Attack! 🚀

A Space Invaders-style browser game with tactical firing, barriers, 
a shop system, and a boss fight. Built with vanilla HTML, CSS, and 
JavaScript — my first vibe-coded game, made with AI assistance.

[Play Bugs Attack! in your browser](https://jarouge1.github.io/Bugs-Attack-/)

## 📥 Download
Grab the latest release to play offline → [Releases](https://github.com/jarouge1/Bugs-Attack-/releases)

## Controls

**Movement**
- ← → Arrow keys to move your turret

**Firing**
- Stop moving to aim
- Z = diagonal left
- X = straight up
- C = diagonal right
- You cannot fire while moving until MOBILE FIRE is unlocked

## Features

**Barriers**
- Two barriers protect you from the swarm
- Bugs are drawn to barriers — they will be targeted
- Barriers block enemies but not bullets — stay sharp
- Repair barriers in the shop between levels

**Gold & Shop**
- Kill enemies to earn Score and Gold (10 pts = 1 gold)
- Spend gold in the ARMORY between levels

**Upgrades**
- MK II → MK III → MOBILE FIRE → APEX
- Each tier improves your firepower — APEX is devastating

**Victory**
- Survive 10 levels and destroy the Millipede boss
